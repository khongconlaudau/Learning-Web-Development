***** Border ******

syntax: 
    1. border: 5px(thickness) solid(style) black(color);

        border-top: 0px; (remove the top border)
    
    2. border-width: 0px 10px 20px 30px; (Clock Direction) (Top Right Bottom Left)

        border-width: 0px 20px;   (top + bottom) = 0px    (right + left) = 20px



    
**** Padding ****  SAME SYNTAX WITH BORDER ABOVE
- Adding the padding or space between the element and border.

sytax: 
    1. padding: 20px;  adding padding 20px to all dimensions






**** Margin ****   SAME SYNTAX WITH BORDER ABOVE
- The part that outside of the border. Adding space between the next element

syntax: 
    1. margin: 10px;    




**** Content Division Element ****
- Like a box contains the content

syntax: <div></div>
